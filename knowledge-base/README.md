# 巴利语翻译知识库

本目录保存从 `translations/` 派生出的翻译知识库。原始 JSONL 文件不在此处修改。

## 结构

- `versions/`: 每个 JSONL 版本的独立知识库。
- `comparative/`: 不同版本之间的用词、风格、句式比较。
- `shared/`: 跨版本共用的语法、策略、模式分类。
- `exports/`: 未来供 RAG 或 GraphRAG 使用的导出结果。
